- relational
  - 发音
    - 英
      - `/rɪ'leɪʃ(ə)n(ə)l/`
    - 美
      - `/rɪ'leʃənl/`
  - 词义
    - adj.
      - 有关系的,亲属的
        - `indicating or expressing syntactic relation, as for example the case endings in Latin `
  - 同根词
    - adj.
      - `relative`
        - 相对的；有关系的；成比例的
      - `related`
        - 有关系的，有关联的；讲述的，叙述的
    - adv.
      - `relatively`
        - 相当地；相对地，比较地
    - n.
      - `relative`
        - 亲戚；相关物；[语] 关系词；亲缘植物
      - `relationship`
        - 关系；关联
      - `relation`
        - 关系；叙述；故事；亲属关系
      - `relativity`
        - 相对论；相关性；相对性
      - `relativism`
        - 相对主义；相对论
      - `relatedness`
        - 关联性；关系；亲缘
    - v.
      - `related`
        - 叙述（relate过去式）
    - vt.
      - `relativise`
        - 相对化（等于relativize）
      - `relativize`
        - 用相对论处理…
  - 近义词
    - adj.
      - 相关的；亲属的
        - `correlative`
        - `pertinent`
  - 短语
    - `relational database`
      - 关系数据库 
    - `relational expression`
      - [计]关系表达式 
    - `relational model`
      - 关系模型 
    - `relational algebra`
      - [数]关系代数 
    - `relational graph`
      - 相关图形；关系图 
  - 例句
    - `One of these tables contains only traditional relational columns, while the second contains relational columns and one XML column.`
      - 这两个表的其中之一仅包含传统的关系列，而另一个包含关系列和一个 XML 列。
